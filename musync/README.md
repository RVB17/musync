# Music Taste Groups App

## Overview
The Music Taste Groups App is a React Native application that allows users to sign into Spotify or Apple Music, collect their listening data to create a music taste profile, and enables users to create and join location-based groups to play music that mixes their shared tastes or fits specific moods or settings.

## Features
- User authentication via Spotify and Apple Music
- Collection of listening data to create personalized music taste profiles
- Location-based music groups for collaborative listening experiences
- Ability to create and join groups based on shared music tastes and moods
- Music player functionality to play, pause, and skip tracks

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/musync.git
   ```
2. Navigate to the project directory:
   ```
   cd musync
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Run the application:
   ```
   npm start
   ```

## Usage
- Upon launching the app, users will be prompted to log in using their Spotify or Apple Music accounts.
- After logging in, users can view their music taste profile and join or create music groups based on their location.
- Users can play music from their groups, which will be tailored to the collective tastes of the group members.
